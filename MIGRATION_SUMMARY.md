## Key Differences

### Platform Detection:
```typescript
// OLD (Cordova)
if (this.platform.is('cordova')) { }

// NEW (Capacitor)
if (this.platform.is('capacitor')) { }
```

### Plugin Imports:
```typescript
// OLD (Cordova) - No imports needed
this.statusBar.styleDefault();
this.splashScreen.hide();

// NEW (Capacitor)
import { StatusBar, Style } from '@capacitor/status-bar';
import { SplashScreen } from '@capacitor/splash-screen';

await StatusBar.setStyle({ style: Style.Default });
await SplashScreen.hide();
```

### Configuration:
```xml
<!-- OLD (Cordova) - config.xml -->
<widget id="com.tejcoms.news">
  <preference name="SplashScreenDelay" value="3000" />
</widget>
```

```typescript
// NEW (Capacitor) - capacitor.config.ts
const config: CapacitorConfig = {
  appId: 'com.tejcoms.news',
  plugins: {
    SplashScreen: {
      launchShowDuration: 3000
    }
  }
};
```

## Benefits of Capacitor

1. **Better Performance** - Native web views (WKWebView on iOS)
2. **Modern Architecture** - Native projects are source controlled
3. **Easier Development** - Better debugging and live reload
4. **Active Support** - Ionic's primary focus
5. **Plugin Ecosystem** - Growing modern plugin library
6. **iOS Support** - Better WKWebView support out of the box
